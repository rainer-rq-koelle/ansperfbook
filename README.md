Getting started for Rainer.

* open new project --> bookdown, choose folder for project
* initalise git repo: usethis::use_git(), choose Yes for initial commit and restart
* hook up local git repo and github: usethis::use_github() (initial push failure: follow recommendation in shell: git push --set-upstream origin master)

tinytex::install_tinytex(dir = "C:\\Users\\rkoelle\\dev\\TinyTeX", force = TRUE)


This is a minimal example of a book based on R Markdown and **bookdown** (https://github.com/rstudio/bookdown). Please see the page "Get Started" at https://bookdown.org/home/about/ for how to compile this example.
