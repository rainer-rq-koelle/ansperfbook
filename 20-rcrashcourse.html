3  Using R

This section provides a short introduction to R. The R-ecosystem provides a rich environment for data analytics. This section offers a crash-course like walk over some basics and the use of the {tidyverse} packages. The goal is provide “enough R” for a newcomer to follow the code and understand to call the provided functions or adapt the analysis template to own needs/interests.
